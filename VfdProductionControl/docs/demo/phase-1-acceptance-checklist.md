# Phase 1 验收清单

使用本清单配合 [Phase 1 演示脚本](phase-1-demo-script.md) 验证当前第一阶段基线。

## 操作员流程

- [ ] 应用可以启动，并默认进入“操作员控制台”。
- [ ] 输入 `EMP0001` 后，操作员会话可以启动。
- [ ] 可以选择 `VFD 生产测试演示方案`。
- [ ] 可以选择 1、2、3 号槽位。
- [ ] 槽位按用户选择顺序提示扫码和绑定。
- [ ] `VFD202605280001`、`VFD202605280002`、`VFD202605280003` 均可通过条码校验。
- [ ] 确认开始后，选中槽位进入运行状态。

## 执行和结论

- [ ] 模拟设备会执行 VFD 启动命令。
- [ ] 模拟流程会读取 VFD 电压。
- [ ] 模拟流程会读取仪表电压。
- [ ] 电压比对会生成通过、失败或警告等明确结论。
- [ ] 最终结论按槽位步骤结果聚合。
- [ ] 失败槽位不会停止其他槽位，除非方案步骤配置为立即停止。
- [ ] 流程完成或失败时会尝试执行停止 VFD 命令。
- [ ] 槽位卡片用颜色和文字共同表达状态。

## 追溯

- [ ] 测试完成后，追溯数据可以查询。
- [ ] 可以按日期范围筛选历史记录。
- [ ] 可以按 VFD 条码筛选设备运行记录。
- [ ] 可以按结论筛选设备运行记录。
- [ ] 设备运行详情包含步骤明细。
- [ ] 步骤详情包含测量数据。
- [ ] 步骤详情包含比对结果。
- [ ] 命令追踪包含请求 JSON 和响应 JSON。

## 工程边界

- [ ] Phase 1 demo 默认使用模拟设备，不依赖真实 COM 口。
- [ ] SQL Server 持久化仍是骨架或可选集成，不阻塞内存模式演示。
- [ ] 真实串口 Modbus 骨架已编译通过，但默认未启用。
